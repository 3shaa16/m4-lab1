# Employee Management Application

A simple web-based application to manage employee records, including adding, viewing, and deleting employees.

## 🚀 Overview

This project was developed as part of an assignment to practice **JavaScript DOM manipulation and data persistence**.
It allows users to interact with employee data dynamically in the browser.

## ✨ Features

* Add new employees through a form
* Display employee data in a table
* Delete employees with confirmation
* Automatically update employee count
* Persist data using browser `localStorage`

## 🛠️ Tech Stack

* JavaScript (50.7%)
* HTML (49.3%)
* Bootstrap 4 (UI styling)

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/3shaa16/m4-lab1.git
   ```
2. Open the folder
3. Run `index.html` in your browser
